# Flow player
A beautiful, cross-platform music player written in Avalonia and C#.

![Flow Player](https://raw.githubusercontent.com/Maciejowski2006/Flow/master/.github/player.png)
The music playing is [ValueFactory @ Furality Umbra](https://youtu.be/YHAv69l-8Tg), and the cover is made by [MaxFilmsVR](https://x.com/MaxFilmsVR/status/1813521301455438204).

### Features
[x] File playback (.aac, .flac, .mp3, .ogg, .opus, .oga, .wav, .wma)
[x] Renders cover art (embedded file, image file with the same name)
[x] Independent audio output device
[] SMTC Support
[] MPRIS (DBus media player controls)
